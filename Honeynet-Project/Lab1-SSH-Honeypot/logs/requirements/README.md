# SSH Honeypot Logs

This directory contains the logs generated by the SSH honeypot (`ssh_honeypot.py`) during its execution. The logs are stored in **JSON Lines (JSONL)** format, where each line represents a unique connection attempt with information like:

- **Timestamp** of the connection attempt
- **Source IP** of the connecting machine
- **Source Port** used for the connection
- **Username** attempted for login
- **Password** attempted for login
- **SSH Banner** sent to the client
- **Additional Data** (if available)

## Log File

- `ssh_honeypot_logs.jsonl`  
  This file contains the logs in **JSONL** format. Each line is a JSON object that records the details of one connection attempt.

### Example Log Entry
```json
{
  "timestamp": "2024-11-25T14:30:00Z",
  "src_ip": "192.168.1.10",
  "src_port": 44320,
  "username": "root",
  "password": "123456",
  "banner": "SSH-2.0-OpenSSH_7.9",
  "extra": "Failed login attempt"
}
