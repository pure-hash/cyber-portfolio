# Lab 2 – Social Media Exposure

## Overview
Identify and analyze sensitive information publicly exposed on social media platforms.

## Objectives
- Assess social media accounts for sensitive data
- Detect potential risks to personal or organizational security
- Document and summarize findings

## Tools Used
- OSINT tools for social media
- Python or Bash scripts
- Manual inspection and screenshots

## Steps
1. Collect publicly available social media data into `/datasets`
2. Analyze accounts for sensitive information
3. Document screenshots in `/images`
4. Summarize exposure and provide mitigation strategies

## Findings
- Sensitive posts or information detected
- Exposure risk evaluated
- Recommendations provided

## Skills Demonstrated
- Social media reconnaissance  
- Risk evaluation  
- Reporting  

## Folder Structure
- **`/datasets`** – Collected social media data  
- **`/images`** – Screenshots of posts or accounts  
- **`/scripts`** – Automation scripts
