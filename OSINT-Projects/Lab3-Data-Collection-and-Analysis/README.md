# Lab 3 – Data Collection and Analysis

## Overview
Collect and analyze publicly available data on a target entity to identify security or privacy risks.

## Objectives
- Gather data from multiple open-source sources
- Correlate and analyze findings
- Document risks and recommendations

## Tools Used
- OSINT search engines
- Data scraping and parsing tools
- Python or Bash scripts

## Steps
1. Identify target entity and sources
2. Collect data into `/datasets`
3. Analyze collected information
4. Capture visuals in `/images`
5. Document findings and recommendations

## Findings
- Collected data summarized
- Risk analysis completed
- Recommendations documented

## Skills Demonstrated
- Data collection and correlation  
- Threat and privacy analysis  
- Reporting  

## Folder Structure
- **`/datasets`** – Collected data for analysis  
- **`/images`** – Visualizations or screenshots  
- **`/scripts`** – Data collection and analysis scripts
