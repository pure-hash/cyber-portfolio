# Lab 1 – Self-Audit

## Overview
Conduct a personal OSINT assessment to understand digital footprint and potential privacy risks.

## Objectives
- Identify personal information available online
- Evaluate exposure and privacy risks
- Document findings and recommendations

## Tools Used
- Google Dorking
- Social media search tools
- Python or Bash scripts for automated data collection

## Steps
1. Search for personal information using OSINT techniques
2. Collect publicly exposed data in `/datasets`
3. Take screenshots of findings in `/images`
4. Analyze and document risks
5. Provide recommendations for mitigation

## Findings
- Sensitive information discovered
- Exposure risks evaluated
- Mitigation recommendations created

## Skills Demonstrated
- OSINT data collection  
- Digital footprint evaluation  
- Reporting and risk analysis  

## Folder Structure
- **`/datasets`** – Collected data  
- **`/images`** – Screenshots of exposure  
- **`/scripts`** – Automated collection or analysis scripts
