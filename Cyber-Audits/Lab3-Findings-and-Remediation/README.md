# Lab 3 – Findings and Remediation

## Overview
Document findings from the audit and provide remediation steps to improve security posture.

## Objectives
- Summarize vulnerabilities and risks
- Provide clear remediation recommendations
- Prepare professional audit report

## Tools Used
- Word/Excel for reporting
- SIEM dashboards or scan results
- Python or PowerShell for automation

## Steps
1. Consolidate findings from previous labs
2. Create remediation plan
3. Capture screenshots or evidence in `/images`
4. Document findings and recommendations in `/datasets`

## Findings
- Security gaps identified
- Remediation steps outlined
- Comprehensive report created

## Skills Demonstrated
- Findings documentation  
- Remediation planning  
- Professional report writing  

## Folder Structure
- **`/datasets`** – Consolidated findings and reports  
- **`/images`** – Visuals supporting findings  
- **`/scripts`** – Optional automation scripts for remediation
